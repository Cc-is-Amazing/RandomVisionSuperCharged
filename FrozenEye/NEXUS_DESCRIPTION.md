# Frozen Eye

## 简介 / Overview

`Frozen Eye` 是一个为《杀戮尖塔 2》制作的轻量功能模组。
启用后，战斗中查看抽牌堆时，卡牌会按照真实抽牌顺序显示，而不是原版默认的按稀有度和 ID 排序。

`Frozen Eye` is a lightweight quality-of-life mod for *Slay the Spire 2*.
When enabled, inspecting the draw pile in combat shows cards in actual draw order instead of the base game's default rarity-and-ID sorted view.

## 功能说明 / Features

- 战斗中查看抽牌堆时，按真实抽牌顺序显示卡牌
- 不改变实际抽牌结果
- 不改变洗牌、随机种子或战斗数值
- 启用模组即可生效，无需额外配置

- Shows the draw pile in actual draw order during combat
- Does not change actual draw results
- Does not alter shuffle logic, RNG seed, or combat balance
- Works immediately once the mod is enabled, with no extra configuration required

## 安装方法 / Installation

1. 完全退出游戏。
2. 将发布包中的 `mods/Frozen Eye` 文件夹复制到游戏根目录下的 `mods` 文件夹中。
3. 启动游戏并确认模组已加载。

1. Close the game completely.
2. Copy the included `mods/Frozen Eye` folder into the game's `mods` directory.
3. Launch the game and confirm that the mod is loaded.

## 使用方式 / How To Use

1. 进入一场战斗。
2. 打开抽牌堆查看界面。
3. 此时看到的顺序就是当前真实抽牌顺序。

如果不想使用该效果，直接在模组菜单中禁用或移除模组即可。

1. Enter combat.
2. Open the draw pile inspection screen.
3. The displayed order will match the current real draw order.

If you do not want this behavior, simply disable or remove the mod.
